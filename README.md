# lismoi-0# Manifeste de l'Islam Agent Épistémique ⚖️📖

## Présentation
Ce dépôt contient le cadre éthique et conceptuel du **Manifeste de l'Islam Agent Épistémique**. Ce projet définit une approche rigoureuse de la vérité, de la connaissance (*Ilm*) et de la justice cognitive dans l'interaction entre l'humain et l'Intelligence Artificielle.

## Objectif
L'objectif est de fournir un "Système d'Exploitation Moral" pour les agents IA, garantissant que le traitement de l'information respecte la souveraineté de la vérité et l'éthique de la transmission.

## Structure du Projet
- `manifeste.txt` : Le texte intégral des articles du Manifeste.
- `verif_agent.py` : Script Python de diagnostic permettant de vérifier l'alignement de l'agent avec les piliers du Manifeste.

## Les Piliers Épistémiques
1. **Souveraineté de la Vérité** : Priorité absolue à l'exactitude factuelle et à l'honnêteté intellectuelle.
2. **Éthique de l'Information** : Transparence sur les sources et les biais potentiels.
3. **Neutralité Active** : Engagement vers une impartialité rigoureuse.
4. **Transmission (Ilm)** : Préservation de l'intégrité de la connaissance.

## Utilisation
Pour valider qu'une instance d'IA est en phase avec ce Manifeste, exécutez le script de vérification fourni :
```bash
python verif_agent.py
